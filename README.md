## 🏛 제 1 전시관: 양자 동적계획법 난독화기 (Quantum DP Obfuscator)

**"관측하는 순간 데이터는 붕괴한다."**

이 코드는 단순한 동적 계획법(DP) 문제가 아닙니다. 
메모리 주소 추적을 무력화하기 위해 고차원 텐서 메모리에 가중치 파편을 흩뿌리며(Smear), 심지어 난수 시드조차 고정하지 않아 **결과값을 관측하려는 순간 완벽한 우주 배경 복사(랜덤 가비지 값)가 출력됩니다.**

- **특징 1:** 파이썬 3의 유니코드 변수명을 악용한 Fraktur(독일 마름꼴) 흑마법 네이밍
- **특징 2:** 파이썬에서 굳이 쓸 필요가 없는 C/C++ 스타일의 분기 없는 조건문 (Branchless Max)
- **특징 3:** 완벽한 양자 얽힘 부재로 인한, 영원히 잃어버린 정답

> *"코드의 의도는 완벽했다. 다만 답이 나오지 않을 뿐."*
